Assignment 1: Simulates a letter guessing game. User is given a certain amount of tries to correctly guess a random letter. Determines whether or not the user's guess is correct

Assignment 2: Creates a shopping menu where the user can access different shopping options, buy items and load their account with a value of money the user inputs

Assignment 3: Simulates the game 'Hangman'. Obscures the correct answer and compares the users' inputs to the correct values. Determines if the user wins or loses based on the number of guesses

Assignment 4: Creates a database that stores information for characters from the game "Super Smash Bros. Ultimate". Starts with some predefined characters. Users can add, view, and delete information for characters currently defined in the database. Data is sorted but the characters' roster number. The information can be printed to a text file.
